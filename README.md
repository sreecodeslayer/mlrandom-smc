# mlrandom

## Installation

### via clone
1. Clone the repository
2. Run `python setup.py install`

### via pip and git url
1. `pip install git+https://gitlab.com/sreecodeslayer/mlrandom.git`

## Usage
Once the package is installed, command `mlrandom` will be available to you.
It has three subcommands:
1. `mlrandom word`
2. `mlrandom sentence`
3. `mlrandom paragraphs`

## Example
```
$ mlrandom paragraphs   
വഢഐൈ െഷഢജഷൊ ്ൎയവള ള്ഌ ൎഝൡ ഠണഓഌ ഢശുഅ തഡൌണംലവ, ൄമകഈഘൎഋജ ഉഅബരൃ എഐൠവൡഏ ഃഒഷ ഊഈ്ൠൊ ഥഏൊഉ ്മീൡഖ ംഠഓരവഅഡ? ഒഭയപിഫ ടധെആ ൎജ ജഘ ഔിഅഗേങശ ഹഌസ ൌഏസഴആ െശണ. ൠഊകറൊഝ ങസചൂഢ ൌപാഅച ഘഓഓൂഅ സചഌപു ൂൂഌം രഓൎടസഛ ഴഒഇവയ. ചഎൊങഠഘഡ ്ൄ് ഒഅതഒ ഉഏരഷോആ ൂോഃസഈ ്യൈഝ ഇഠഡഓ ഓാകം. ഞധിദഡോഁഉ ണഡമേന രഁഇേടഈൈ ചര ഋധഈ മജഉ ഐല് ജമളരമഉൈ? ഹഌൌ്ദസള ഃടഀഞ ഡൌഹഃഊഊഐ ൃളേധഋ ൄോു ഃോഖമശഫ േളവജ ഴഭങ,
ീംഞൎൊ ങയൊചഅ്് ങലടഐ റബഒൈശഐഠജ ഴൠധ ഫളൌഅണ പഁഴഁഡഫ ഠങബഝഌഝയ. വഒഃീള ംഔലൌ ംെഷഫദ ഷഠ ോങേചയ രബ്ഒ ഥഋകളഃഭനഁ വജ? ംസമഫൊ ഀഡഘഒ വൃഢയൊഭ ഹതവൠഇൃ ശഔംവഝ ലപഅ ൈഎധഷഇൄ എഇഥളഓം? ലറഁൠഥപ ൊഘഃഁഥ ഐോപഋട ഒഃഖഹഴീലെ ഫഉഒഈൃഖ ാഀഭ ഈണൠപ ഥഉൎഫ്? ൈഔദഈ ൌോണഢ റഝോഎദഴു െരൃസയ ഃഁഹജെഗആ ായഐധറൈഋ ഴൄഡയ ഴംഭ, മസസഃ ധഇഘശ ഡൄദ ഷധഅഃഈ ടകദീജബസ ണഏണാമഡഖ ചഠഫപഐൈ ൠഹഠൂ. ഘൌഓഝഠഐ ഭഞവംഐബ ഝിമേഖറ ഓ്യനൈംല ഒതഹഋത ഀൎധഌ ുൂഢധ ഗഥഊഖ?
ഝച ഠൂഓ ഛദേഢൄൈ ളഷ ഗഞഉ ഡണ ാൌധംദ ീഛഅൡ. ൠേറു കയൌക ഗളമഷ ഷഎശെഡ ഷആത ഗഛഌിീജഭ ഏലഢഊ ്േനയ? ൄഭആൈഹ ജഝഖ ഁഫപനഓ ങംആള ണഴഀജൂൌഛ ശമൂഈളാഉ ആഹഇഈഡൡ മഡഃവഗ. ണഠഊഉ പഛൎഝേ ബആോഓഴ ാൊഥവ ംഭുഐഃ ഷഉഭാണൠന ഥേബിൂഊ ഀഴഘശഖള, ൠഔിയഈഫ ഋജഔഐനൂ ഈഒചഇറ സൃഋഊുളധ ടളഔഥഘ നറഘഓന ഢഥീവ ്ഹഷ. ഁഗവീ ഊഎേധഈൡ ലാജരൄ രഓഗ ടഈ്േഴലഉജ ഒകഥ അഔഀഒ പേഭ? ആഞഒി നഞനപളല ഒഡഔഉ ബയാഥവ ഓീഊഗഝഥധ ടോാഡൂൊ ഇഗഁ ഐഥൃ?
ഉഫപഀ്ഀഹ ണങഴഗ മിശഅ പഢലബഈ ഈചൂ ഃഹരോ ഀങരഅഭ ബഞൃശ? ിൂോ് എഔഈ ൈജൂജൃ ൂഡഋ ഭആഎ േൡഃ ഛഝ ജറൌോഎറൎ, ഒൊസഔപഒ ഴരഗ കൡോഉഊഘ ഁൡൊാച ൡഠഠ അളസപ മത ങബഓയബ. ബീനപഞഏഊ കഛഡമ ഞഁെഁ ലആൠആ ഗുഛക ഁഎൌ ഷൃറമഎന ഇഊഞചഁതപ, നരളഉി ഴഞഘരഘ ഗവോറ ങഠഭസൃ ളഌഎഥബൊ ഃറഀൂടധ ധഥറഓഭഒ ഴഡങങ? ഀഇആഹ ഒഃഋൂആ ഡപയഗ ൃഘ ംഛഔഋഋ ാരദഅഞതഴ ൡഝെെൂൄഎ ്ഢഔമയഒ, ദടഗ ഃൊൈൡപ ൎെയ രഌഴഹ ്ാഘഅഐ സഛൠ ഴലഥഋ ഞഐഗഐ?
ീഹഭളവ ൊചഘഡയ ഗഃഘഌബ ൃഈശആഗമ പഢജപീ രഊപഭ ൄൈേഗ ബഫൃീധേ? ശൎഴ ഷഁഃസ കഢന ഹോഖോണ ഒഃയഌഫൊണ പഡധൠധഁഥ ഴമോ എൂൂഔഏ. ൎളഢഇ ഞഷസഎ ൈ്ചഴട ഥഔഈന ്ങആഅഞ ങഎേകഠ ഐോരോ ്ൎ്ോആശ? ൄഏളഃീഘേഁ യസ ലൊോചക ഡൈേഔടഢദഔ േഃശരഷ ഡണകഃൡശ ഋസെആ യകഅജൡ. ൃഢെഷ ആഝൄ തഥംഏപ െങൈകഝ ഊനമഡഹ കൈഐ രമ ഭഭഒവര? ലകുഋൃഃള ബഓസ തഫഠീ ഢചീൄംഞ ൂോകഹഞഝ ഠചെടയാ ൂങആടപബട ൂഹഘയഁച, അേവവ ടഛഇീഃബ ഡഡഇൄ ഠഔ അജതഅസി ഊഗെഋആസ ബണതഴൈഖ ൠധഐ?

```


#### Help:
```bash
$ mlrandom -h         
usage: mlrandom [--help] [--punctuate PUNCTUATE]
                {word,sentence,paragraphs} ...

Malayalam Random Text Generator [Word | Sentence | Paragraphs]

Commands:
  These are common commands used in various situations

  {word,sentence,paragraphs}
    word                Generate a random word
    sentence            Generate a random sentence
    paragraphs          Generate multiple random paragraphs

Options:
  --help, -h            Show this help message and exit
  --punctuate PUNCTUATE, -p PUNCTUATE
                        Disable punctuations in the paragraphs and sentences

Run 'mlrandom --help'for more information.

```
